READ ME FOR FINAL PROJECT BST

In this project, this is a fully implemented Binary Search Tree using Java.

It can:
- insert nodes 
- delete nodes 
- search if a node is in the tree
- Depth (Postorder, Preorder and Inorder) and Breadth first search 
- Give the height of the tree
- Give the min value, the max value, and the nth largest value
- Print the root node


This project was made for CMPT-220L-903-21S Software Dev I course at Marist College.
